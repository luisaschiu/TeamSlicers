◆ task_motor1()

void task_motor1 ( void *  param)

Task for the pusher motor.

Handles all actions for the pusher motor. Reads shares changed by the user interface webserver. Based on User interface, the motor is pushing for a certain period of time, then stops. It then waits for the blade task to finish the cut, then moves. This repeats until the pusher hits the limit switch at the end of its travel.

◆ task_motor2()

void task_motor2 ( void *  param)

Task for the blade motor.

Handles all actions for the blade motor. Reads shares changed by the user interface webserver. The motor is capable of homing itself when asked to by the user. It then waits for the pusher task to finish its motion, then moves and slices the produce. This repeats until the pusher hits the limit switch at the end of its travel.

◆ task_ultrasonic()

void task_ultrasonic ( void *  param)

Task for ultrasonic sensor.

Ultrasonic measures the distance between the blade and the start of the object to be sliced. Pusher brings up the produce until it trips the ultrasonic sensor.
